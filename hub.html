Jak to podpiąć pod GitHub Pages?
1) W każdym repo (arkon-crm i pawel) włącz Settings → Pages, Source: Deploy from a branch, Branch: main / (root).
2) Ten plik (hub.html) też wrzuć do repo z włączonym Pages (np. hub).
3) Podaj swój login GitHub (małymi literami) poniżej — linki ustawią się automatycznie.
4) Domyślne adresy będą miały postać: https://USERNAME.github.io/arkon-crm/ oraz https://USERNAME.github.io/pawel/
